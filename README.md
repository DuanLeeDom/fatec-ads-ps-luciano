# Repositório de Aulas de Programação de Script

Bem-vindo ao repositório das minhas aulas de **Programação de Script**! Este repositório foi criado para organizar e armazenar os materiais, exercícios e projetos que desenvolvo durante meu estudo de linguagens de programação voltadas para a web, incluindo **HTML**, **CSS** e **JavaScript**.

## Sobre o Repositório

O objetivo principal aqui é documentar meu progresso enquanto aprendo os fundamentos e técnicas dessas três tecnologias essenciais para o desenvolvimento web:
- **HTML**: Estruturação de conteúdo na web.
- **CSS**: Estilização e design visual de páginas.
- **JavaScript**: Interatividade e dinamismo nas aplicações web.

Este repositório serve como um portfólio pessoal de aprendizado, onde pretendo aplicar o que estudo em projetos práticos e compartilhar meu desenvolvimento ao longo do tempo.

## Estrutura do Repositório

- **`aulas/`**: Contém os arquivos das aulas organizados por tópicos ou datas.
- **`exercicios/`**: Exercícios práticos para reforçar o aprendizado.
- **`projetos/`**: Pequenos projetos que combinam HTML, CSS e JavaScript.
- **`recursos/`**: Materiais de apoio, como links, PDFs ou anotações.

## Tecnologias Utilizadas

- **HTML**: Para criar a base das páginas web.
- **CSS**: Para estilizar e tornar os projetos visualmente atraentes.
- **JavaScript**: Para adicionar funcionalidades interativas.
